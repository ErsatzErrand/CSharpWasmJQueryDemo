C# WASM JQuery Wrapper Prototype
WASM Package: package_2bd82c7aebbda97bc86bfb7963b6c1b9a1fa2505

This demo serves as a proof-of-concept of creating a managed C#WASM wrapper for JQuery, which in turn facilitates DOM manipulation from C#WASM.

We setup buttons which make JS->C# interop calls, and the C# type responds in turn by making C#->JQuery calls to further manipulate the DOM. The C# type has a simple counter to track how many clicks have been made, and also holds a collection of rows.

All of the C# is running client side inside the browser as WebAssembly. For additional information and source code see: https://github.com/ErsatzErrand/CSharpWasmJQueryDomInteropt
